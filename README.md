# BrainBoard

BrainBoard is an AI-powered study workspace designed for students to manage tasks, organize notes, and summarize lectures in one place.

The goal of BrainBoard is to create a calm and focused environment where students can track their academic progress and stay organized.

---

## 🚀 Live Demo
https://brainboard-calm-hub.lovable.app

---

## ✨ Features

- Task management
- Notes organization
- Lecture summaries
- Clean productivity dashboard
- AI-powered study workspace

---

## 🛠 Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite

---

## 📦 Installation

Clone the repository:

git clone https://github.com/siri488/brainboard-

Install dependencies:

npm install

Run the project:

npm run dev
